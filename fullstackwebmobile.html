Full Stack Web and Multi-platform Mobile App Development

Build Complete Web and Hybrid Mobile Solutions. Master front-end web, hybrid mobile app and server-side development in five comprehensive courses.
Curriculum provided by Coursera and Hong Kong University of Science and Technology
Learn front-end and hybrid mobile development, with server-side support, for implementing a multi-platform solution.
The first two courses in this curriculum cover front-end frameworks: Bootstrap 4 and Angular. You’ll also learn to create hybrid mobile applications, using the Ionic framework, Cordova and NativeScript. On the server side, you’ll learn to implement NoSQL databases using MongoDB, work within a Node.js environment and Express framework, and communicate to the client side through a RESTful API.
​
Learners enrolling in this curriculum are expected to have prior working knowledge of HTML, CSS and JavaScript, which can be acquired by joining our introduction course first.

Courses
​Front-End Web UI Frameworks and Tools: Bootstrap 4 (4 weeks)
Commitment: 4 hours/week online lectures and exercises, 3 hours/week in-person workshop and assignments.
Cost of the individual course: $240
This course will give you an overview of client-side web UI frameworks, in particular Bootstrap 4. You will learn about grids and responsive design, Bootstrap CSS and JavaScript components. You will learn about CSS preprocessors, Less and Sass. You will also learn the basics of Node.js and NPM and task runners like Grunt and Gulp. You will also learn how to use source control tools like Git and GitHub.

​At the end of this course, you will be able to: 
  1. Set up, design and style a web page using Bootstrap 4 and its components,
  2. Create a responsive web page design, and
  3. Make use of web tools to setup and manage web sites.

This course also enables you to work on your own project developing a website using Bootstrap 4.
​Students enrolling in this course should have prior good working knowledge of HTML, CSS and JavaScript.
Front-End JavaScript Frameworks: Angular (4 weeks)
Commitment: 6 hours/week online lectures and exercises, 4 hours/week in-person workshop and assignments.
​Cost of the individual course: $240
​This course concentrates mainly on Javascript based front-end frameworks, and in particular the Angular framework (Currently Ver. 4.x). This course will use Typescript for developing Angular application. Typescript features will be introduced in the context of Angular as part of the exercises. You will also get an introduction to the use of Angular Material and Angular Flex-Layout for responsive UI design. You will be introduced to various aspects of Angular including components, directives and services. You will learn about data binding, Angular router and its use for developing single-page applications. You will also learn about designing both template-driven forms and reactive forms. A quick introduction to Observables, reactive programming and RxJS in the context of Angular is included. You will then learn about Angular support for client-server communication and the use of REST API on the server side. You will use Restangular for communicating with a server supporting the REST API. A quick tour through Angular animation support and Angular testing rounds off the course. You must have either completed the previous course in the specialization on Bootstrap 4, or have a working knowledge of front end web-UI frameworks to be able to navigate this course. Also a good working knowledge of JavaScript, especially ES 5 is strongly recommended.

At the end of this course you will:
  1. Be familiar with client-side Javascript frameworks and the Angular framework
  2. Be able to implement single page applications in Angular
  3. Be able to use various Angular features including directives, components and services
  4. Be able to implement a functional front-end web application using Angular
  5. Be able to use Angular Material and Angular Flex-Layout for designing responsive Angular applications
  6. Be able to use Observables and RxJS in the context of Angular applications
Multiplatform Mobile App Development: Ionic and Cordova (4 weeks)
Commitment: 6 hours/week online lectures and exercises, 4 hours/week in-person workshop and assignments.
​Cost of the individual course: $260
This course focuses on developing multiplatform mobile applications using the Web technologies (HTML5, CSS and Javascript). In particular we make use of the Cordova hybrid application framework to develop and target multiple mobile platforms with a single codebase. We make use of the Ionic framework (Ionic Ver 3.x), one of the most popular mobile application frameworks, that is built with mobile-optimized HTML5 and CSS based components and Angular. You will learn about UI development with Ionic and then using Cordova's modules to access the native mobile platform's capabilities from Javascript. You should have already completed the Bootstrap 4 and the Angular courses in this specialization before proceeding with this course.

At the end of this course you will be able to:
  1. Build mobile applications targeting multiple platforms with a single codebase,
  2. Leverage your HTML5, CSS, Javascript and Angular skills, and
  3. ​Use various features of the Ionic framework to build hybrid mobile applications
Multiplatform Mobile App Development: NativeScript (4 weeks)
Commitment: 6 hours/week online lectures and exercises, 4 hours/week in-person workshop and assignments.
​Cost of the individual course: $260
This course focuses on developing truly cross-platform, native iOS and Android apps using NativeScript (Ver 3.x). The framework uses Angular, TypeScript or modern JavaScript to get truly native UI and performance while sharing skills and code with the web. You will learn about UI development with NativeScript UI and layout support and access the native mobile platform's capabilities from Javascript. You should have already completed the Bootstrap 4 and the Angular courses in this specialization before proceeding with this course.

At the end of this course you will be able to:
  1. Build mobile applications targeting multiple platforms with a single codebase,
  2. Leverage your Angular, TypeScript and Javascript skills, and
  3. Use various features of the Ionic framework to build truly cross-platform mobile applications
Server-side Development with NodeJS, Express and MongoDB (4 weeks)
Commitment: 6 hours/week online lectures and exercises, 4 hours/week in-person workshop and assignments.
​Cost of the individual course: $260
This course deals with all things server-side. We base the entire course around the NodeJS platform. We start with a brief overview of the Web protocols: HTTP and HTTPS. We examine NodeJS and NodeJS modules: Express for building web servers. On the database side, we review basic CRUD operations, NoSQL databases, in particular MongoDB and Mongoose for accessing MongoDB from NodeJS. We examine the REST concepts and building a RESTful API. We touch upon authentication and security. Finally we review backend as a service (BaaS) approaches, including mobile BaaS, both open-source and commercial BaaS services.

At the end of this course, you will be able to:
  1. Demonstrate an understanding of server-side concepts, CRUD and REST
  2. Build and configure a backend server using NodeJS framework
  3. Build a RESTful API for the front-end to access backend services
Your learning experience
You will join a classroom of 20 local students, led by an expert instructor.

You will watch online lectures and perform exercises during the week, which should take up to 3 to 5 hours, on your own schedule.
If you have any question during that time you will be able to engage with the classroom and the instructor using a private group, and the instructor will make sure you get the help you need.
​At the end of each week you will be asked to join a 3 to 4 hours in-person  workshop to work with your classroom on the end-of-week assignment. This is also when you'll start to work on your own portfolio project.
You will work in smaller teams of 2 to 4 students, and will learn and collaborate together throughout the course and curriculum. You will also be asked to review other team's work and offer feedback.

Frequently Asked Questions
What to expect after registering?
Each curriculum is starting on a Friday evening. Two weeks prior that date you will be invited to meet with your fellow classmates and instructor for a one hour kick-off workshop (in-person), during which you will be introduced to the tools you will need for the duration of the curriculum.

What is the typical weekly experience?
Every Friday evening you will receive specific instructions including the lectures and exercises to perform during the next week, including a description of the final assignment. You will checkout the learning collaboration tool and discuss with the instructor and your fellow students about your project daily, and get your questions answered there. At the end of the week (either Friday evening or Saturday morning) you will meet in-person for a 3 to 4 hours workshop to work on the weekly assignment as well as your portfolio project.

What happens if I need to take break during the curriculum?
We hope that you will find the time to stay committed to the curriculum. In the event you need to take a break we would recommend you do so at the end of one of the 5 courses, and before starting the next one. We will then reschedule you to the next delivery, where you left off. A rescheduling fee of $50 will be applied.

Can I get a refund if I decide to not finish the curriculum?
We will issue a full refund a week prior to the start of the curriculum, and no refund after that. We would recommend registering for the individual courses if you're unsure about committing to the full curriculum, or use the rescheduling option indicated above to take a break.

Can I look at the curriculum content prior to registering?
Yes, the curriculum is delivered in partnership with Coursera. You can check it out here.

What background knowledge is necessary?
You need to have a good working knowledge of HTML, CSS and JavaScript. We offer a 3 week course to help you with that, check it out here.

How are you able to offer such a low price? Are you cutting on quality?
Quite the contrary actually. Our business model is different and allows us to offer better quality overall. We leverage the content already developed by renowned professors, instead of building our own. We have developed a network of cafe to host the in-person workshops, instead of leasing buildings and classrooms. We invest these savings on hiring quality instructors, and back to you with super low prices.  That approach also allows us to deliver our programs closer to where you live.
